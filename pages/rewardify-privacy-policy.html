1. Introduction

Rewardify is a loyalty and rewards platform that helps Shopify merchants increase customer retention and engagement through points-based rewards, VIP tiers, and personalized incentives. We are committed to protecting customer privacy and ensuring transparency in our data practices.

Our Commitment:

  • We collect only the data necessary to provide loyalty program functionality
  • We never sell customer data to third parties
  • We use industry-standard security to protect all data
  • We provide full GDPR compliance for EU customers
  • We automatically delete all data when merchants uninstall our app

2. What Data We Collect

2.1 Customer Profile Data

We collect the following customer data from Shopify to provide loyalty program functionality:

Data Field Purpose Required
Customer ID Unique identifier for loyalty account Yes
Email Address Account identification, notifications, cross-store tracking Yes
First Name Personalized communications Optional
Last Name Personalized communications Optional
Phone Number SMS notifications (if enabled) Optional
Order History Points calculation, tier progression Yes

2.2 Loyalty Program Data

We create and manage the following loyalty-specific data:

  • Points Balance: Current available points for redemption
  • Lifetime Points Earned: Total points earned historically
  • Lifetime Points Redeemed: Total points spent on rewards
  • Current Tier: Customer's VIP tier level (Bronze, Silver, Gold, etc.)
  • Tier Progress: Progress toward next tier level
  • Enrollment Date: When customer joined the loyalty program
  • Referral Code: Unique referral code for referring friends
  • Referral Count: Number of successful referrals

2.3 Transaction Data

We track all loyalty-related transactions including:

  • Points earned from purchases
  • Points earned from actions (referrals, sign-ups, etc.)
  • Points redeemed for rewards
  • Tier upgrades and downgrades
  • Expiration events

4. Data Storage & Security

4.1 Multi-Tenant Data Isolation

Each merchant's customer data is completely isolated from other merchants:

  • All database queries include shopId filter for data segregation
  • Shopify session token authentication required for all API requests
  • No cross-merchant data access is technically possible

4.2 Security Measures

5. Data Retention & Deletion

5.1 Automatic Data Deletion on Uninstall

Automatic Deletion: When a merchant uninstalls Rewardify, ALL customer data associated with that store is permanently deleted from our servers within 24 hours.

This includes:

  • Customer loyalty profiles
  • Points balances and transaction history
  • Tier information
  • Referral data
  • All associated records

5.2 Data Retention Period

While the app is installed, customer data is retained:

  • Active customers: Retained indefinitely while app is installed
  • Uninstalled stores: All data deleted within 24 hours
  • Transaction history: Kept for points calculation and audit trail

5.3 Customer-Initiated Deletion

Customers can request deletion of their loyalty data at any time through:

  • Your store's privacy settings (via Shopify's customer data request system)
  • Contacting the merchant directly
  • Contacting Rewardify support
